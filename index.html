About

The project "Binocoli" consists of designing an in-depth thematic section of the "Corago" portal focused on comparing different versions of opera libretti. The operas examined are Verdi's “Macbeth”, Rossini's “Tancredi” and Donizetti's “La Favorita”. These operas were respectively written and edited by Piave Francesco Maria and Maffei Andrea, Rossi Gaetano and Lechi Luigi, Royer Alphonse and Vaëz Gustave (Jannetti Francesco as a translator and Bassi Calisto) linked by the common theme of the modified ending. The project intends to develop a user interface that enriches Corago's offerings through increased interactivity and user involvement in comparing different versions of the texts. The project portal will offer the user both a section dedicated to the historical context about the inception and the editorial development of the operas, considered through textual and critical lenses, and a geographical/chronological navigation tool that can offer the user a direct experience with the changes of the contexts.
